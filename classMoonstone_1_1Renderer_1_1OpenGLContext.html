void Moonstone::Renderer::OpenGLContext::Init ( )
virtual

Initializes the OpenGL context.

Sets the current context to the specified GLFW window and loads OpenGL functions using GLAD. Logs OpenGL information such as vendor, renderer, and version.

Note
This function asserts that the GLAD loader was successful.
