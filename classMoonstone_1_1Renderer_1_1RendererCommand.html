◆ s_RendererAPI

RendererAPI* Moonstone::Renderer::RendererCommand::s_RendererAPI
staticprivate

Pointer to the RendererAPI instance.

This static member points to an instance of RendererAPI, which is initialized based on the preprocessor directives. The actual instance depends on whether MS_OPENGL or MS_VULKAN is defined.

Note
This is only defined if either MS_OPENGL or MS_VULKAN is set.
